# some error codes :
### error codes for this format : 
    example :
    "detail": "Authentication credentials were not provided.",
    "code": "not_authenticated"

#### 401 :
    while page need auth !
        "code": "not_authenticated"
        
    while refresh or access token is expired
        "code": "token_not_valid",
        
#### 400  (sometimes):
    while page need auth !
        "code": "bad_request"
